Spam Detection ML Model

I developed a spam detector project in Python using a pipeline approach consisting of custom transformers. The project begins with an EmailToWordCounterTransformer class, which preprocesses email text by stripping headers, converting to lowercase, replacing URLs and numbers, removing punctuation, and optionally applying stemming. This transformer demonstrates text preprocessing techniques crucial for natural language processing tasks. The next step utilizes WordCounterToVectorTransformer to convert word counts into feature vectors, limiting the vocabulary size to 10 most common words per email. This transformer prepares data for my logistic regression model by vectorizing text into sparse matrices, showcasing feature engineering skills. Together, these components form a robust pipeline for email spam detection, illustrating proficiency in data preprocessing, feature extraction, pipeline construction, and model selection.
